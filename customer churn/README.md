# Customer-Churn-prediction
Customer Churn Prediction App
Overview

Customer churn is one of the most critical challenges faced by subscription-based businesses. This project is a Machine Learning-powered web application that predicts whether a customer is likely to leave a telecom service based on customer demographics, account information, and service usage patterns.

The application is built using the Kaggle Telco Customer Churn dataset and provides real-time churn predictions through an interactive user interface.

Problem Statement

Customer retention is generally more cost-effective than acquiring new customers. By identifying customers who are at risk of churning, businesses can take proactive actions such as personalized offers, customer support interventions, and loyalty programs.

This project aims to:

Predict customer churn using historical customer data.
Analyze key factors influencing churn.
Provide a simple interface for business users to make predictions.
